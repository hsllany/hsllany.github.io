非科班出身如何成为 UI 设计师？

2015-02-11

知乎问题：非科班出身如何成为 UI 设计师？

其实科不科班真的并不太重要。我理科出身，学了6年应用数学专业，最后转到了用户体验这个行业里

我的体会是：

1、UI设计是一个非常靠实践的工种，不太依赖于理论基础。凡是这类的工作，科不科班出身，并不太重要。这不像计算机等学科，需要从计算机基础、编程语言、数据结构等学起。新手到大牛这中间的门槛，并不在于理论基础，而是大量的实践和经验。当然，我不是说理论不重要，但理论掌握的多少，并不能从根本上决定一个设计师的好坏。所以，从这个角度上来说，你只要把实践时间堆够了，这个差距是能弥补的；

2、UI设计要靠一点灵性。很多科班出身的人，也不一定能做好UI设计，或设计类的工作。这是因为，审美、创意，这些东西是很难靠实践来弥补的，或者很难靠3、4年的大学实践或学习来提高的。这有时候就是一种天分，一种从小就培养起来的感觉和习惯。所以，如果你觉得，比起其他同学或朋友，自己很容易能分辨什么是美、什么是漂亮、很容易对视觉或体验的东西产生敏感、很容易产生想法，那么不要犹豫，这行就很适合你。

至于说如何提高和学习，通俗的说，其实就是边学边干。把你70%的时间花在实习，做项目、做设计上，争取到找工作时有满满的案例，丰富的经验；20%的时间花在理论学习上，如果觉得自己的理论基础薄弱，适当看一些这方面的书籍，当然注意，切记变成只会纸上谈兵的“伪专家”，目标是熟知各种概念和方法，并尝试在工作中运用；另外，花10%来多认识相关行业的人，和他们交流等；

当然，这其中，实习是一道坎，没经验不太好找实习，不实习又没经验，这似乎是个伪命题。但你还是可以通过学校、学生会的项目、私活、以及实现自己想法的设计等途径来突破。上培训班这种途径，只能领你进门，不会有本质的区别，如果学费便宜还是可以考虑。
